# Wazuh Server Setup (Ubuntu)

## Objective
Deploy a fully functional Wazuh SIEM server.

---

## Steps Performed
- Installed Wazuh on Ubuntu Server
- Enabled and validated:
  - Wazuh Manager
  - Wazuh Indexer
  - Wazuh Dashboard

---

## Validation
- Verified services were running
- Confirmed dashboard accessibility via browser
- Checked indexer and manager health status

---

## Outcome
A stable SIEM platform ready to ingest logs from multiple endpoints.
