## Overview
This repository documents a hands-on cybersecurity lab where I deployed **Wazuh SIEM** integrated with **Suricata IDS/IPS** to achieve centralized security monitoring across Windows and Linux endpoints.

The project simulates a real-world SOC environment and demonstrates skills relevant to **SOC Analyst, Cybersecurity Analyst, and GRC roles**.

---

## Project Objectives
- Deploy a functional SIEM from scratch
- Centralize logs from Windows and Linux endpoints
- Detect host-based and network-based security events
- Visualize alerts and logs via a security dashboard
- Practice SOC-style alert triage and analysis

---

## Environment Summary
- **SIEM Platform:** Wazuh
- **IDS/IPS:** Suricata
- **Server OS:** Ubuntu
- **Endpoints:** Windows, Ubuntu
- **Dashboard Access:** Web GUI (Chrome)

---

## Architecture Overview
- Wazuh Server (Manager, Indexer, Dashboard)
- Windows Endpoint with Wazuh Agent
- Ubuntu Endpoint with Wazuh Agent and Suricata IDS/IPS
- Centralized log ingestion and alerting

See: `architecture/architecture-overview.md`

## Key Skills Demonstrated
- SIEM deployment and configuration
- Endpoint onboarding and monitoring
- IDS/IPS integration
- Alert validation and triage
- Security monitoring aligned with GRC principles
